# R Package Dependencies

## Required packages
- MASS (mvrnorm for multivariate normal generation)
- nnet (multinom for multinomial logistic regression)
- MplusAutomation 1.1.1 (Mplus interface)
- parallel (mclapply for parallelization; fork-based, macOS/Linux only)
- dplyr (data manipulation)
- tidyr (data reshaping)
- ggplot2 (figure generation)
- gridExtra (figure layout)
- grid (figure layout)

## R version
- R 4.4.3 (tested; requires >= 4.0.0)

## External software
- Mplus 8.3 (tested; requires >= 8.0, must be on PATH)
